# Milestone v1.5: Report Formatting

**Status:** SHIPPED 2026-01-20
**Phases:** 27
**Total Plans:** 2

## Overview

Enable users to customize how list items appear in their generated radiology reports with per-section configuration.

## Phases

### Phase 27: Report List Style Preferences
**Goal**: Users can customize bullet/list styles for report sections via Settings
**Requirements**: FMT-01, FMT-02, FMT-03, FMT-04, FMT-05, FMT-06, FMT-07, FMT-08
**Plans**: 2 plans (complete)

Plans:
- [x] 27-01: Settings UI for list style preferences
- [x] 27-02: Apply list styles to report generation and exports

---

## Milestone Summary

**Key Accomplishments:**
- Settings page "Report Formatting" section with list style preferences
- 5 list styles available: Bullet (•), Dash (-), Arrow (→), Numbered (1.), None
- Per-section configuration (Clinical Info, Findings, Impression, etc.)
- "Apply to All" button for quick global style changes
- Preferences stored in user_preferences table
- Styles applied to report generation
- PDF export renders list styles correctly
- Word export renders list styles correctly

**Stats:**
- 1 phase, 2 plans
- User preference driven formatting
