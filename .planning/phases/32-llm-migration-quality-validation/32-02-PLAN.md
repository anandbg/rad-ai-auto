1. Create `app/lib/ai/quality-validation.ts` with these validation functions:

   **validateReportStructure(report: string): StructureResult**
   - Check that all 5 required sections exist as `## SectionName` headings
   - Sections: Clinical Information, Technique, Comparison, Findings, Impression
   - Return: `{ valid: boolean; missingSections: string[]; foundSections: string[] }`

   **validateSubheadings(report: string, expectedSubheadings: string[]): SubheadingResult**
   - Extract all `**bold text:**` patterns from the Findings section
   - Compare against expectedSubheadings from the fixture
   - Return: `{ adherencePercent: number; missing: string[]; found: string[]; extra: string[] }`
   - Pass threshold: >95% adherence

   **validateAntiHallucination(report: string, inputConcepts: string[], findings: string): HallucinationResult**
   - Extract all abnormal findings from the report (non-normal, non-template statements)
   - Check each abnormal finding mentions at least one concept from inputConcepts
   - Flag findings that introduce measurements, sizes, or specifics not in the original findings text
   - Return: `{ clean: boolean; suspectedHallucinations: string[]; conceptsCovered: string[] }`
   - This is heuristic-based: flag suspicious patterns like specific measurements (e.g., "2.3 cm") that don't appear in the input

   **validateImpressionFormat(report: string): FormatResult**
   - Check Impression section uses bullet points (lines starting with "- ")
   - Return: `{ valid: boolean; bulletCount: number; nonBulletLines: string[] }`

   **runFullValidation(report: string, fixture: TestFixture): ValidationReport**
   - Runs all 4 checks above
   - Returns aggregate result with pass/fail per check and overall pass/fail
   - Overall pass requires: structure 100%, anti-hallucination clean, subheading adherence >95%

2. Export all types and functions. Use pure functions with no side effects. No external dependencies beyond the test-fixtures module.

3. Include JSDoc comments explaining each validation function's purpose and thresholds.
