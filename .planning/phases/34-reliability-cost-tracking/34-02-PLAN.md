<objective>
Add a provider fallback chain so that when Groq fails (error, timeout, rate limit), the system automatically retries against OpenAI GPT-4o. Wire fallback into the three text-generation routes (generate, templates/generate, templates/suggest). Replace hardcoded cost tracking in these routes with real AI SDK usage data so cost reflects whichever provider actually served the request.

Purpose: Research flagged 30+ Groq outages in 5 months. Without fallback, a Groq outage = total service failure. With fallback, users transparently shift to OpenAI at ~30x cost, so cost tracking must record the amplification (otherwise the daily ceiling, COST-01, can't protect the budget during an outage).

Output: New `app/lib/ai/fallback.ts` module + updates to 3 route handlers to use it and the new provider-aware trackCost.
</objective>

    Create `app/lib/ai/fallback.ts`:

    ```typescript
    import { registry } from './registry';
    import { getModelId } from './config';
    import { createLogger } from '@/lib/logging/logger';
    import type { LanguageModel } from 'ai';

    const logger = createLogger('AIFallback');

    export const FALLBACK_CHAIN = {
      generate: 'openai:gpt-4o',
      template: 'openai:gpt-4o',
      transcription: 'openai:whisper-1',
    } as const;

    export interface FallbackResult<T> {
      result: T;
      modelId: string;
      fellBack: boolean;
    }

    export async function withProviderFallback<T>(
      purpose: 'generate' | 'template',
      operation: (model: LanguageModel, modelId: string) => Promise<T>
    ): Promise<FallbackResult<T>> {
      const primaryId = getModelId(purpose);
      const fallbackId = FALLBACK_CHAIN[purpose];

      try {
        const primaryModel = registry.languageModel(
          primaryId as Parameters<typeof registry.languageModel>[0]
        );
        const result = await operation(primaryModel, primaryId);
        return { result, modelId: primaryId, fellBack: false };
      } catch (primaryError) {
        // Don't fallback if primary IS the fallback (avoid infinite loop and double-cost)
        if (primaryId === fallbackId) {
          throw primaryError;
        }

        logger.warn(
          `Primary ${primaryId} failed, falling back to ${fallbackId}: ${
            primaryError instanceof Error ? primaryError.message : String(primaryError)
          }`
        );

        try {
          const fallbackModel = registry.languageModel(
            fallbackId as Parameters<typeof registry.languageModel>[0]
          );
          const result = await operation(fallbackModel, fallbackId);
          return { result, modelId: fallbackId, fellBack: true };
        } catch (fallbackError) {
          logger.error(
            `Both providers failed. Primary (${primaryId}): ${
              primaryError instanceof Error ? primaryError.message : primaryError
            }. Fallback (${fallbackId}): ${
              fallbackError instanceof Error ? fallbackError.message : fallbackError
            }`
          );
          throw fallbackError;
        }
      }
    }
    ```
