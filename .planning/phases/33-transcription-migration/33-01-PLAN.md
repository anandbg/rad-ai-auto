**Step 1: Create `app/lib/ai/medical-vocabulary.ts`**

Create a new file exporting a `RADIOLOGY_VOCABULARY_HINT` string constant. This string is passed as the `prompt` parameter to the Whisper API to bias transcription toward medical terminology. Research found Groq Whisper mis-transcribes "dexamethasone taper" 37% of the time without hints.

The hint string should contain representative radiology terms separated by commas, covering:
- Anatomical terms: pneumothorax, pneumomediastinum, atelectasis, effusion, consolidation, emphysema, cardiomegaly, hepatomegaly, splenomegaly, lymphadenopathy
- Imaging sequences: T1-weighted, T2-weighted, FLAIR sequence, DWI, ADC map, STIR, post-gadolinium, fat-saturated
- Modality terms: MRI, CT, X-ray, ultrasound, mammography, fluoroscopy, PET-CT
- Medications/contrast: dexamethasone, gadolinium, iodinated contrast, metformin, heparin
- Measurements: Hounsfield units, SUV, millimeters, centimeters
- Common findings: herniation, stenosis, fracture, metastasis, aneurysm, thrombosis, embolism, calcification, ossification
- Spine terms: L4-L5, C5-C6, T12, foraminal, facet, ligamentum flavum, disc desiccation

Keep the hint under 500 characters total (Whisper prompt parameter has a practical limit). Format as a single string of comma-separated terms -- no sentences.
